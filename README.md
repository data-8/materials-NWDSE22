## File Structure

- `lec02regression/`: Regression Lecture
  - `lec02.ipynb`: Regression Lecture Main Demo
- `lab02regression/`: The Regression Lab
  - `lab02.ipynb`: Main Lab assignment
  - `tests/`: Autograder tests
  - Other associated supportive files
